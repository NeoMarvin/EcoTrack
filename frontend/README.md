This is the client-side application for EcoTrack. It provides a responsive and interactive user interface for users to manage their eco-journey.

# Technologies

React (Vite)

Tailwind CSS

React Router DOM

Axios

React Context API (Auth & Activity State)

# Features

Responsive Landing Page: Modern UI with hero section and feature highlights.

Authentication Flows: Login and Registration forms with error handling and redirects.

Protected Dashboard: Only accessible to logged-in users.

Dynamic Navbar: Updates based on login state.

Activity Management: Real-time adding and deleting of eco-actions.

 Run the Client

# Install dependencies
npm install

# Start the development server
npm run dev


The application will be available at http://localhost:5173.
