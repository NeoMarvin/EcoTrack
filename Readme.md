EcoTrack is a comprehensive web application designed to help individuals track their personal environmental impact. Users can log eco-friendly actions (like recycling, planting trees, or saving energy), view their history on a private dashboard, and visualize their contribution to a greener planet.

This project was built as a final capstone project for the PLP Full-Stack Development Program.

JWT (JSON Web Tokens) (Authentication)

The project is divided into two main directories:

frontend/: Contains the React client application, pages, components, and styles.

‚ö° Quick Start Guide

To run this application locally, you will need two separate terminals.

1. Setup the Backend

Open a terminal and navigate to the backend folder:

cd backend


Install dependencies:

npm install


Create a .env file in the backend folder and add your variables:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key


Start the server:

npm start


Server will run on http://localhost:5000

2. Setup the Frontend

Open a new terminal and navigate to the frontend folder:

cd frontend


Install dependencies:

npm install


Start the React app:

npm run dev


Client will run on http://localhost:5173
